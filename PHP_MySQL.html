¿Qué es PHP y para qué se utiliza?

PHP (Hypertext Preprocessor) es un lenguaje de programación de código abierto diseñado para el desarrollo web. Se utiliza principalmente para crear páginas dinámicas e interactivas, conectarse a bases de datos y manejar formularios. PHP se ejecuta en el servidor, generando contenido HTML que se envía al navegador.

¿Qué programas requiero para poder utilizar PHP?

Un servidor web (como Apache o Nginx). El intérprete de PHP. Un editor de texto o entorno de desarrollo (como VS Code o Sublime Text). Opcionalmente, un gestor de bases de datos como MySQL. Puedes instalar todos estos componentes juntos usando un paquete como XAMPP, WAMP, o MAMP.

¿Qué es un servidor web?

Un servidor web es un software o hardware que entrega contenido a través de internet. Este procesa solicitudes HTTP y sirve archivos (como páginas HTML o respuestas generadas por PHP) al navegador del usuario.
